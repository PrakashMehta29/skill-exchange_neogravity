# 🌟 SkillSwap: Hyperlocal Skill Exchange
A community-driven, hyperlocal skill exchange platform where neighbors trade knowledge instead of money. Built with a premium, native-app feel using Vanilla JS and CSS Glassmorphism.
## 📖 Project Overview
**SkillSwap** is a peer-to-peer learning network designed to foster community growth by enabling users to trade skills without financial transactions. Instead of paying for a masterclass, you can teach someone how to bake sourdough bread in exchange for them helping you with your taxes.
The platform was built with a strong focus on premium UI/UX, featuring a native-feeling application architecture, zero-dependency Lottie-style CSS animations, and a secure, password-manager optimized login flow.
## ✨ Key Features
- **Interactive Radar Scanner:** Uses browser geolocation to simulate scanning the local area (within 5km) for nearby neighbors ready to trade.
- **Zero-Knowledge Trade Verification:** Users watch short, interactive video clips of a skill and must pass a Knowledge Check Q&A quiz to verify learning and earn community coins.
- **Learning History Dashboard:** Automatically aggregates and tracks your total session time and completed learning modules.
- **Native Device Sharing:** Utilizes the Web Share API to trigger the operating system's native share sheet (iOS/Android) for viral community invites.
## 🚀 Technical Stack
- **Frontend:** Vanilla HTML5, CSS3, JavaScript (ES6)
- **Architecture:** Single Page Application (SPA) with mock state management.
- **Styling:** Custom Glassmorphism UI, CSS Keyframe Animations.
- **Icons:** Phosphor Icons.
- **Deployment:** Ready for Firebase Hosting.
## 🛠 Setup & Run Locally
Since this is a lightweight frontend application, no complex build steps are required.
1. Clone the repository.
2. Open `index.html` in your favorite modern browser.
3. Use any email and password to log in and explore the mock state environment!
